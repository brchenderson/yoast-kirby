Kirby SEO with Yoast

Requires Kirby 2.3

Version 0.01

This is a fork of Kirby SEO. It adds in Yoast SEO content analysis to any content field, Open Graph and Twitter previews and a group of fields to drop into the site.yml for default meta title, description Open Graph image and Twitter image.

To get the meta fields add this to a blueprint:

  meta:
    extends: meta

The plugin looks for a field called text by default but you can add a contentblockname field to a blueprint to analyze something with a different name. Example:

  seo:
    label: Kirby SEO
    type: seo
    contentblockname: somethingotherthantext
